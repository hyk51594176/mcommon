# 介绍

mcommon 是基于elemen-ui封装的vue组件，MSelect、MForm、Mtable，灵活的配置，强大的功能，能够大大提高前端同学的开发效率

# 安装

  npm install @hanyk/mcommon

import Vue from 'vue'
import elementUI from 'element-ui'

import mcommon from '@hanyk/mcommon'
Vue.use(elementUI)
Vue.use(mcommon)
